Welcome to the ChannelAdvisor Application by Mozu

ChannelAdvisor is powerful, cloud-based eCommerce software that connects retailers and manufacturers to multiple online sales channels, including Amazon, Google, eBay, and more. The ChannelAdvisor Application by Mozu syncs your Mozu products, orders, and customers with any number of third-party marketplaces to optimize sell-through and customer reach. You can map your Mozu sites to different online marketplaces to take advantage of alternative sales channels and clear out excess inventory.

Application Features:

  • Map sites on your Mozu tenant to one or many ChannelAdvisor marketplaces.
  • Map one or many Mozu locations to your ChannelAdvisor Distribution Center to manage inventory across platforms.
  • Schedule regular order imports from ChannelAdvisor to ensure timely fulfillment.
  • Map your Mozu attributes (standard and custom) to the ChannelAdvisor attributes of your choice.
  • Force sync data as needed for real-time updates.

Install the ChannelAdvisor Application on Your Tenant

The ChannelAdvisor Application by Mozu is provided free of charge on GitHub. Clone the Mozu/ChannelAdvisor repository and use the provided gradle script to build the application. Refer to the readme file included with the GitHub repository for more information about building and installing the ChannelAdvisor application.

Note: Contact Mozu Support if you cannot access the Mozu/ChannelAdvisor repository on GitHub.

Configure the Application

Configuration Requirements

Review the following requirements to ensure you have everything you need to be successful, then complete the steps in this section:

  • The ChannelAdivsor Application must be installed on your tenant. Refer to the Readme file on GitHub for preqrequisites to installing the application.
  • You must have a ChannelAdvisor developer account.
  • You must be a ChannelAdvisor partner.

Launch the App Configuration Settings

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. In the Applications list, click Channel Advisor R5.
  3. Click the Configuration link to open configuration settings.

The remainder of this section describes actions you complete on tabs of the configuration settings dialog.

Authorize Your ChannelAdvisor Account

The ChannelAdvisor Applicaton uses your ChannelAdvisor Profile ID to link Mozu and ChannelAdvisor. You can get this ID from

  1. In the app configuration settings dialog, go to the Authorization tab.
  2. Enter your ChannelAdvisor Profile Id in the text box. If you don't know your ID, you can get it from the ChannelAdvisor Account Dashboard:
    1. Log in to ChannelAdvisor.
    2. From the menu on the top right, select the ChannelAdvisor account you want to authorize:
    3. From the top bar of the Account Dashboard, select My Account > Company Setup. The Profile ID appears at the top of your Company Information.
  3. Wait for ChannelAdvisor to approve the authorization. Once it is approved, you can continue with configuration.

Map Your Mozu Sites to ChannelAdvisor Marketplaces

You can map each site on your Mozu tenant to a separate marketplaces in ChannelAdvisor, and you can map a single site to multiple marketplaces:

  1. In the app configuration settings dialog, go to the Product tab and click Marketplaces. Every site on your Mozu tenant appears in the Select Marketplace list.
  2. In the Channel Advisor Marketplace column, enter the name of the marketplace to map to the corresponding Mozu Site. The textbox uses type-ahead to suggest marketplaces, and displays all standard ChannelAdvisor marketplaces. You can also enter a custom marketplace name if you have specified one in ChannelAdvisor.
    Note: The marketplace you select must be a valid, configured marketplace in ChannelAdvisor.
  3. (Optional) To map the same Mozu site to additional marketplaces, use a comma and a space to separate each marketplace name.
  4. When you are done mapping marketplaces, click Save. Any sites you do not map to marketplaces do not send or recieve data from ChannelAdvisor.

Map Product Attributes

While Mozu and ChannelAdvisor share many similar product attributes, each platform contains certain attributes that the other does not, and both platforms allow you to create custom attributes. Therefore, it is important to configure your product mappings before you export products from Mozu to ChannelAdvisor.

  1. In the app configuration settings dialog, go to the Product tab and click Attributes.
  2. Select the site you want to map from the drop-down menu. A mapping table with three columns appears: Certain standard mappings are pre-populated, but you can change them to anything you want. Note that all products in ChannelAdvisor must have a Title and a SKU.
  3. For each Mozu attribute you want to send to ChannelAdvisor, enter a mapping in the Mapping CA Attribute column. The textbox suggests standard ChannelAdvisor attributes, but you can also enter any custom attributes you have created. Just make sure the name you enter matches the attribute name in ChannelAdvisor exactly.

    Note: You are not required to map every Mozu attribute to ChannelAdvisor. You can leave a field empty if you do not require that data on other marketplaces.

  4. Click Save.
  5. Repeat this procedure for each site you mapped to a marketplace in ChannelAdvisor. You can map the same Mozu attributes to different ChannelAdvisor attributes for different Mozu sites.

Map Locations for ChannelAdvisor Inventory

To ensure timely fulfillment, ChannelAdvisor must know current inventory levels for your Mozu products. The ChannelAdvisor Application handles this by mapping Mozu locations to ChannelAdvisor Distribution Centers so that both systems are updated when order information is processed. You can map multiple Mozu locations to a single ChannelAdvisor Distribution Center. ChannelAdvisor uses the combined inventory or all mapped Mozu locations as its total available inventory.

  1. In the app configuration settings dialog, go to the Product tab and click Location.
  2. Select the Mozu locations you want to use for ChannelAdvisor inventory.
  3. Enter the code for the ChannelAdvisor Distribution Center you want to use for your Mozu inventory. All your Mozu products must map to a single Distribution Center in ChannelAdvisor.
  4. Click Save.

Inventory is synchronized every time you import orders from Channel Advisor and process them in Mozu.

Enable the App

Complete the following steps to enable the ChannelAdvisor Application by Mozu:

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. Click Channel Advisor R5.
  3. Click Enable App.

Congratulations! You can now use sell your Mozu products on third-party marketplaces through ChannelAdvisor.
