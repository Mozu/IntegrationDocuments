October 2015

Updated documentation for Mozu Import/Export Tool. Enhancements include:

  • Extended template file tables to include full descriptions and valid values for each column.
  • Added documentation for Orders template.
  • Added documentation for new ProductTypeAttributeValues sheet in Products template.

July 2015

Thanks for using the Mozu Import/Export Tool! Here's what's new in this version:

  • Fixed a bug that caused attribute exports to fail for attributes of a list type which contained integer values.

March 2015

Thanks for using the Mozu Import/Export Tool! Here's what's new in this version:

  • Added internationalization support. The app supports international sites with the following:
    • Locale values of Product Attribute Name
    • Locale values of Product Attribute Vocabulary
    • Locale values of Category Name
    • Locale values of Product Variant
    • Locale values of Product properties
    • Locale currency values of Product Extras
  • Enhanced Usability
    • Image uploading now independent from associating an image with a product.
    • Merged Products/Product catalog tabs for use under Single MasterCatalog/Single Catalog case. You can still find category codes within the Products tab.
    • Added ability to upload any single tab or combination of tabs. This app supports the following:
      • Products
      • Product Catalogs
      • Products
      • Product Images
      • Product Variants
      • Product Extras
      • Product Variants Locale (only available for multi-currency tenants)
      • Product Properties Locale (only available for multi-locale tenants)
      • Categories
      • Location Inventory
      • Images
    • Added ability to upload same category sheets multiple times without loading duplicates.
    • Added ability to select which columns to update. Only Master Catalog Name, Product Code, and Catalog Name(if it exists) are required.
    • Added support formulas in Exel cells. For example, you can set the sale price to be 10% in your Excel data.
  • Performance Improvements
    • Imports are faster because image uploading no longer requires associating the image with a product.

January 2015

  • Added documentation for the Import/Export tool.
