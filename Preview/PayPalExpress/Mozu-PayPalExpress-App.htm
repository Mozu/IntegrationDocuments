Welcome to PayPal Express and Mozu

PayPal is one of the world's largest digital payment processors, serving over 169 million customers in more than 200 markets. The PayPal Express Integration for Mozu enables you to easily add PayPal's Express Checkout functionality to your existing Mozu payment options.

Note: The PayPal Express for Mozu Integration requires that your site be updated to the Mozu Core 8 theme.

Application Features:

  • Adds PayPal as an additional payment option to your shopping cart and checkout pages.
  • Connects Mozu to your existing PayPal merchant account.
  • Allows customers to expedite purchasing by clicking through a simple PayPal dialog.
  • Imports all necessary order information back into Mozu for payment capture and fulfillment.
  • Works in conjuction with all other payment gateways, including Pay with Amazon.

Install the PayPal Express Integration for Mozu on Your Tenant

Mozu offers two options for acquiring the PayPal Express Integration:

  1. Purchase the PayPal Express Certified Mozu Application. A Mozu sales or professional services representative will install the application on your tenant.
  2. Build the application from the Mozu-provided code on GitHub and install it to Mozu Dev Center. This option requires that you have Arc.js enabled on your tenant. Contact your sales or professional services representative for more information about enabling Arc.js.

Update Your Mozu Theme

Installing and configuring the PayPal Express app enables PayPal Express functionality in Mozu Admin. However, to enable PayPal functionality on your storefront so that customers can use is, you must also merge theme changes to enable the PayPal workflow:

  1. Confirm that your site has been updated to use the Mozu Core 8 Theme. You can find the latest version of the Mozu Core theme on GitHub.
  2. Navigate to the PayPalExpress-Theme repository on GitHub.
  3. Review and apply the changes made in our sample implementation to your own theme.
    Tip: To see the differences between the sample implementation and your own theme, open the command line and run the git diff command, using the paths for the themes to be diffed.

For more information on working with themes, including modifying, uploading, installing, and applying a theme to your site, refer to the Theme Development Quickstart.

Configure the Application in Mozu Admin

Now that your Pay with Amazon application is installed, go to your Mozu Admin tools to configure the integration.

  1. Go to Settings > Payment & Checkout.
  2. Check the PayByAmazon checkbox. Once checked, the various configuration fields appear.
  3. Select the Amazon environment you wish to use. Note that this selection has nothing to do with Mozu production or sandbox environments; this option selects the Amazon environment.
    • Select Sandbox if you wish to use Amazon’s sandbox environment for testing your Pay with Amazon integration.
    • Select Production to use a working Amazon environment that processes payments.
  4. Obtain the below values from Amazon Seller Central (https://sellercentral.amazon.com/):
    • Seller ID
    • Client ID
    • Applicaton ID
    • AWS Access Key
    • AWS Secret
  5. Select an AWS region.
  6. Select an order processing method:
    • Authorize and Capture on Order Placement: This method authorizes and captures payment when an order is placed.
    • Authorize on Order Placement and Capture on Order Shipment: This method authorizes a payment when an order is placed, but does not capture that payment until the order has shipped.
  7. Click Save to save your Pay with Amazon settings.
