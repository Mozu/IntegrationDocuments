Welcome to the Mozu Import-Export Application

App Version: 2.0.0

The Mozu Import-Export Application enables you to easily import and export data - from large coupon sets to detailed IT analytics information - to and from Mozu.

IT Analytics Data

The Mozu platform collects a wide variety of real-time information about your site that can be useful for IT analytics. The Mozu Import-Export Application exports this information - including error incidents, performance metrics, and unavailable links - to a bucket in an Amazon Simple Storage Service (S3) account or to an FTP or FTPS server.

Application Features:

  • Collect real-time information about your site(s) that helps you monitor errors, warnings, performance metrics, unavailable links, and uptime issues.
  • Export logged data to your Amazon S3 account for storage and analysis.
  • Export logged data to an FTP server of your choice.
  • Automate data exports to send updated logs at specified time intervals.

Coupons and Coupon Sets

In Mozu, a coupon is a code that a user enters to enable a discount. Coupons are organized into coupon sets that define the behaviors of their member coupons. Mozu Admin gives you the ability to manually enter specific coupon codes one-by-one or to generate a large number of randomized coupon codes. However, in some cases, you might want to create a large number of fully custom coupon codes. For example, you might want to create HAPPYBIRTHDAYJOHN! codes for every customer in your database. The Mozu Import-Export Application simplifies and expedites that process.

Application Features:

  • Import large sets of custom coupons via a simple CSV template.
  • Export information about coupons and coupon sets to evaluate the effectiveness of various promotions.
  • Specify an email address to receive notifications when import/export operations complete.
  • Queue multiple operations to break up large jobs.

Install the Mozu Import-Export Application on Your Tenant

After purchasing the Mozu-Import Export Application, call your sales or professional services representative and request installation on your tenant. Mozu staff will install the app to your tenant, making it available for configuration and usage in your storefront.

Configure the Application

Note: If you only want to use the Coupons and Coupon Sets functionality of the Mozu Import-Export Application, you can skip to Enable the App. If you want to export IT Analytics data for your Mozu sites, you must complete the steps in this section.

Configuration Requirements

Review the following requirements to ensure you have everything you need to be successful, then complete the steps in this section:

  • The Mozu Import-Export Application must be installed on your tenant.
  • You must have an Amazon S3 account or access to an FTP server to receive the logged data from your site.

Where do you want to log your data?

  • Amazon S3 Account
  • FTP Server

Note: You can configure both export locations if desired, but you can only export data to one destination type at a time.

Configure Export to an Amazon S3 Account

Create a Bucket in Your S3 Account

The Mozu Import-Export Application requires a bucket in your S3 account for the log files. The application recognizes any existing buckets, but you might want to create a unique one just for your Mozu data.

  1. Log in to Amazon Web Services.
  2. Create a bucket for your log files.

You can refer to the Amazon Web Services help for a walkthrough of creating a bucket.

Obtain Amazon S3 Security Credentials

The Mozu Import-Export Application requires your Amazon S3 Access Key ID, Secret Access Key, and active Security Token Service Region.

Note: You cannot view the Secret Access Key on your Amazon account after the initial creation process, so if you have lost or forgotten it, you will have to create new access keys for your account.

  1. Navigate to the Security Credentials page under your account dropdown.
  2. (Optional) If you don't know your Secret Access Key, go to the Access Keys section and click Create New Access Key.
  3. Copy your Access Key ID and Secret Access Key.
  4. Navigate to the Account Settings page.
  5. Under Security Token Service Regions, note your active regions.

Enable Your S3 Account in Mozu

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. Click Mozu Import/Export (version 1.1.0).
  3. Click the Configuration link to open configuration settings.
  4. On the Storage Options tab, select Enable S3 Account and enter the account info you copied in the previous section.
  5. Click Save.

Configure Export to an FTP Server

Obtain FTP Server Credentials

If you want to export logs to an FTP server, note the following details about your FTP setup:

  • Server name
  • Username
  • Password
  • Whether the server uses Secure FTP
  • Port number
  • A directory on the server in which to store the logs of exported data

If you need help obtaining these details, contact your IT department or FTP administrator.

Configure the Mozu Import-Export Settings

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. Click Mozu Import-Export Application (version 2.0.0).
  3. Click the Configuration link to open configuration settings.
  4. On the Storage Options tab, select Enable FTP and enter the information for your FTP server.
  5. Click Save.

Enable the App

If you only want to export IT analytics information, you do not need to complete additional steps to enable the Mozu Import-Export Application. If you want to use the app to import or export coupons and coupon sets, complete the following steps:

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. Click Mozu Import-Export Application (version 2.0.0).
  3. Click Enable App.

Enabling the Mozu Import-Export Application adds an Extensions menu to the Marketing module in Mozu Admin. You will use the Import/Export extension to manage the import and export of coupon sets.

Export IT Analytics Data Logs to S3 or FTP

Use the configuration page of the Mozu Import-Export Application to schedule exports of IT data about your site. You can also view your export history.

Schedule an Export

  1. On the Mozu Admin top bar, select Settings > Applications.
  2. Click Mozu Import/Export (version 1.1.0).
  3. Click the Configuration link to open the configuration settings.
  4. On the Web Logs tab, select Settings.
  5. Use the External Storage dropdown to select the type of export destination.
    Note: You cannot export data to both an S3 and an FTP destination simultaneously. If you want to switch between destinations, you must stop an export and configure a new one.
  6. Using the Export Interval (minutes) dropdown, choose how frequently to export data.
  7. (Amazon S3) Using the S3 Bucket dropdown, choose an Amazon S3 bucket to receive the logs in.
  8. (FTP) In FTP Directory, enter the directory on the FTP server, preceded by a "/", to receive the logs.
  9. Click Save.

Note: If you want to change these settings after configuring an export, stop the export first and then make the changes.

View Logs and Export History

To view full log information, access your Amazon S3 account or FTP server and open the logs.

To check what exports the Mozu Import-Export Application has generated, use the Web Logs page to view a history of exported logs. You can review the date and time of a log and verify whether the export completed successfully or not. If you want to view additional information about an export, click View in the appropriate row.

Stop an Export

You can stop an export at any time:

  1. Open the Web Logs page.
  2. Click Stop Export.

Coupons and Coupon Sets

When you enable the Mozu Import-Export Application, Mozu adds an Extensions menu to the Marketing console in Mozu Admin.

Note: If you just enabled the app and you do not see the Extensions menu on your Marketing console, refresh the browser window.

Selecting the Import/Export extension launches a tool that you can use to import and export coupon and coupon set information.

Understand the Coupons and CouponSets CSV Files

The Mozu Import-Export Application requires specifically named and formatted CSV files that are compressed into a ZIP file. To ensure you start with properly formatted CSV files, navigate to the Import tab of the Import/Export extension and click the link below Choose File to download template files with the required input fields.

Note: The Coupons and CouponSets CSV files that the Mozu Import-Export Application uses contain columns that map to properties in the Coupons and CouponSets resources in the Mozu API.

Coupons.csv

Corresponding API Resource: commerce/catalog/admin/couponsets/coupons

The Coupons file defines individual coupons and maps them to catalogs and coupon sets. Coupons and coupon sets are catalog-level items in Mozu. You cannot create coupons on a master catalog.

Note: The Import-Export Application expects all catalog names to be unique on your tenant. If multiple catalogs share the same name, you must temporarily rename the catalog you want to modify before you can import coupons or coupon sets. You can revert to the original name after the coupons are imported.		 Yes
Coupon Set Code The unique identifier for the coupon set that contains the coupon. All coupons must belong to a coupon set.

If you are creating a new coupon set by importing a CouponSets file with this file, this code can be any string, provided both files use the same Coupon Set Code. CouponSets.csv

Corresponding API Resource: commerce/catalog/admin/couponsets

The Coupon Sets file defines coupon sets in Mozu, including the dates they are valid and the maximum number of times their member coupons can be redeemed. Coupons and coupon sets are catalog-level items in Mozu. You cannot create coupon sets on a master catalog.

Note: Only Mozu can create generated coupon codes. You cannot import generated codes, because specifying the code names in a file is an inherently manual operation. If you are using the Import/Export Application to update existing generated codes, you must export the codes, delete them via Mozu Admin, and then re-import them to a new coupon set as manual codes.

Import New Coupons or Coupon Sets

  1. On the Marketing console in Mozu Admin, select Extensions > Import/Export to display the Marketing Import/Export console.
  2. Select the Import tab.
  3. In the File Format dropdown, select CSV. Currently, CSV is the only supported file type.
  4. Click Choose File to select the file(s) you want to import. You can import just coupons, just coupon sets, or both. Mozu expects a ZIP file that contains one or two CSV files named "Coupons" and "CouponSets", respectively. Your files must follow this naming convention to import successfully.
  5. (Optional) Specify an email address to receive notification when the import is complete. This is useful for very large data sets that might take some time to import.
  6. Click Import.
  7. (Optional) Go to the Jobs tab to view the progress of your import. When the job completes (or if the job encounters an issue), you can click the View link under Logs to get more information about the import operation.

Export Coupons or Coupon Sets

  1. On the Marketing console in Mozu Admin, select Extensions > Import/Export to display the Marketing Import/Export console.
  2. Select the Export tab.
  3. In the File Format dropdown, select CSV. Currently, CSV is the only supported file type.
  4. Specify whether you want to export Coupon Sets, Coupons, or both.
  5. (Optional) Specify an email address to receive notification when the export is complete. This is useful for very large data sets that might take some time to export.
  6. (Optional) To save time and create more readable files, you can filter the export data by specific coupon set codes. Type a code in the Coupon Set Code(s) field and hit <Enter>. The code appears as a checkbox on the Import/Export console. Enter and select as many codes as you want to include in the export file.
  7. Click Export.
  8. (Optional) Go to the Jobs tab to view the progress of your export. When the job completes (or if the job encounters an issue), you can click the View link under Logs to get more information about the export operation.

Queue Multiple Import/Export Operations

You can queue multiple import/export operations to execute in sequence. This is especially useful if you are importing or exporting very large data sets, because you can split a large job into smaller batches and more quickly see whether each small job succeeds.

  1. Decide the order in which you want to process your jobs.
  2. Use the Import/Export console to enter each job, starting with the first you want completed and continuing sequentially.
  3. (Optional) For multiple imports, ensure that you have multiple unique ZIP files with properly formatted, unique Coupons and CouponSets CSV files. Mozu doesn't require a specific name for the ZIP file, so you can give each ZIP a unique name to help identify which part of the job it corresponds to.
  4. After you configure all your imports and exports, go to the Jobs tab in the Import/Export console to see the progress of each individual job.

Note: If a job appears as Pending on the Jobs tab but does not show a Date, you must manually schedule the job by clicking the Schedule button that appears in this situation.
