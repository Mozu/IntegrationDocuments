Welcome to the Mozu Import-Export Application

The Mozu Import-Export Application collects a wide array of information about your site useful for IT analytics, including error incidents and unavailable links. The Mozu Import-Export Application logs this important information to help you troubleshoot issues and maximize uptime, and allows you to send the logs to an Amazon Simple Storage Service (S3) account or to an FTP server.

Application Features:

  • Collect information from your site(s) that helps you monitor errors, warnings, performance metrics, unavailable links, and uptime issues.
  • Export logged data to your Amazon S3 account for storage and analysis.
  • Export logged data to an FTP server of your choice.

Install the Mozu Import-Export Application on Your Tenant

After purchasing the Mozu-Import Export Application, call your sales or professional services representative and request installation on your tenant. Mozu staff will install the app to your tenant, making it available for configuration and usage in your storefront.

Configure the Application

To integrate the functionality of the Mozu Import-Export Application with your site, review the configuration requirements to ensure you have everything you need to be successful with this guide, and then complete the steps in this section.

Configuration Requirements

  • The Mozu Import-Export Application must be installed on your tenant.

(Optional) Obtain Amazon S3 Security Credentials

If you want to export logs to an Amazon S3 account, you need to note the security credentials for the account:

For AWS account access keys (root users), after you sign up for AWS, you must create access keys for the account. You can retrieve the access key ID from the Security Credentials page, but you cannot retrieve the secret access key. If the secret access key is lost or forgotten, you need to create new access keys. You can create new access keys for the account by going to the Security Credentials page. In the Access Keys section, click Create New Access Key.

Configure the Mozu Import-Export Settings

  1. On the Mozu Admin top bar, select Settings»Applications.
  2. Click Mozu Import-Export Application.
  3. Click the Configuration link to open configuration settings.
  4. Open the Options tab.

Enable the App

You do not need to complete additional steps to enable the Mozu Import-Export Application. You can disregard the Enable App button.
